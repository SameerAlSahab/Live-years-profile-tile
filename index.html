How to Use This Age Counter

  • Fork this repository to your GitHub account
  • Create a file named birthdate.txt in the repository root
  • Add your birthdate in the format DD/MM/YYYY
  • Enable GitHub Pages for your repository
  • Add this to your profile README:
    [![My Age](https://yourusername.github.io/repo-name)](https://github.com/yourusername/repo-name)

Note: Months and days can be single or double-digit (e.g. 5 or 05 for May)
